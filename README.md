# desktopAgent

A starter repository for the desktopAgent project.

## Structure

- `src/desktop_agent.py` - starter Python module
- `react-ui/` - React UI scaffold
- `.gitignore` - ignores common Python and editor artifacts plus node_modules
- `requirements.txt` - dependency placeholder

## Usage

Run the starter module:

```powershell
python src\desktop_agent.py
```

## Notes

On Windows, file names are case-insensitive, so this repository uses `README.md` as the primary documentation file.
